# Blood Donor Database System
 Overview
This project is a simple Python program that stores and manages blood donor information.  
It allows adding, viewing, updating and deleting donor records.

 Features
- Add a new donor  
- View all donors  
- Update donor details  
- Delete donor information  
- Stores data inside a list (in-memory database)

 How to Run
1. Open terminal or command prompt  
2. Navigate to the folder where main.py is saved
